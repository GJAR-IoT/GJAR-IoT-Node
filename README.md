# GJAR IoT Project
## Hardware Nodes

Code in this folder (`GJAR_IoT/Hardware/*`) is the source for our Internet of Things nodes.

### Dependencies

- ESP8266WiFi
- ESP8266HTTPClient
- ArduinoJson
- SimpleDHT
- DallasTemperature
- LiquidCrystal_I2C.h
- DS3231
- EEPROM
- MsTimer2
